Modplug tracker (OpenMPT) OpenMPT is an open-source audio module tracker for Windows. It was previously called ModPlug Tracker, and was first released by Olivier Lapicque in 1997.


Famitracker FamiTracker is a free windows tracker for producing music for the NES/Famicom-systems. The interface is based on MadTracker.


Deflemask DefleMask is a cross-platform and multi system chiptune music tracker. (This Is Not Free)
